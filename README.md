## 1. 服务器序列号获取 ➡️ ServerSerialNumberGet

1. 此脚本主要用于从大量服务器中获取序列号。
2. 本脚本文件使用要求：

    * 服务器端已经安装 dmidecode
    * Python 基础环境
    * 安装 paramiko 第三方库
  
3. 缺点：json 文件格式有要求，如果是采用预先创建文件的方式，可能需要先处理格式，较为麻烦。
